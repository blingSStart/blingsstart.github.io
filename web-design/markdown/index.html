MarkDown入门

  2019-11-27

我的github （HbuilderX）markdown常用快捷键总结

  • 空行或者俩次空格加回车换行
  • 文字大小h1-7 + tab
  • 无序列表 li + tab
  • 有序列表 数字num + . + 空格 或者 ctrl +alt +1
  • 加粗字体 ctrl+B
  • 倾斜字体 i + tab
  • 超链接 a + tab *126邮箱*
  1. 分割线 hr + tab
  2. 图片 img + tab
  3. 代码域 code + tab
  4. time + tab 自行理会
  5. 表格 举例：table 3 * 3 + tab
  6. 注意：如果使用区块>在后面加有序或者无序列表时最好保留空格
