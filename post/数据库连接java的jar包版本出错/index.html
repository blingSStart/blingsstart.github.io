Fri, Dec 13, 2019

解决方式

<dependency>
		<groupId>mysql</groupId>
		<artifactId>mysql-connector-java</artifactId>
		<version>5.1.6</version>
	</dependency>
改为：
<dependency>
			<groupId>mysql</groupId>
			<artifactId>mysql-connector-java</artifactId>
			<version>5.1.44</version>
		</dependency>

connector版本号与mysql的版本不一致不一致，详情可查看mysql官方文档

不会的知识不要盲目的查询，了解基本错误信息后一定要查看官方文档，了解根本原因，以防下次的错误
