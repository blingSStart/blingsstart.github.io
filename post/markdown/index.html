MarkDown 简易使用指南

希望能够大佬能给一些建议，正苦恼博客样式太简单（近期才入坑）

markdown是可以使用简单的文本编辑器编写的标记语言，正因如此他的易读易写收到了广泛的青睐，下面就来简单介绍一下采用快捷键

（HbuilderX）markdown常用快捷键总结

  • 空行或者俩次空格加回车换行
  • 文字大小h1-7 + tab
  • 无序列表 li + tab
  • 有序列表 数字num + . + 空格 或者 ctrl +alt +1
  • 加粗字体 ctrl+B
  • 倾斜字体 i + tab
  • 超链接 a + tab *126邮箱*
  1. 分割线 hr + tab

  2. 图片 img + tab

  3. 代码域 code + tab

  4. time + tab 自行理会

  5. 表格 举例：table 3 * 3 + tab

  6. 注意：如果使用区块>在后面加有序或者无序列表时最好保留空格
